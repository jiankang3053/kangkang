# KangkangWeather v3.1.0

## 重点更新

- 控制台左上角新增固定版本号显示，方便远程排查时确认运行版本。
- 新增“运行总览”，集中显示自动发送环境、息屏/电源状态、定时任务状态、下一次完整发送和补偿发送数量。
- 新增运行追踪记录：手动发送、测试消息、固定发送会写入步骤时间线，便于定位卡在哪一步。
- 新增迁移包导出：可生成发给其他电脑的配置包，并强制新电脑重新验证微信目标。
- 新增 GitHub 最新版本检查接口和控制台按钮。
- 打包配置补充 v3.1 运行模块，便携包和安装包都包含运行追踪、迁移包、更新检查能力。

## 使用建议

- 发给别人电脑前，先导出迁移包，不直接复制本机 `%APPDATA%` 配置。
- 新电脑首次运行后先做“兼容性自检”“打开会话测试”“发送测试消息”，再开启自动化固定发送。
- 息屏运行时必须保持 Windows 用户登录且电脑不睡眠；可以关闭屏幕，但不能锁屏后注销或进入睡眠。

## 已知限制

- 微信发送仍依赖官方 Windows 微信客户端的前台 UI 自动化，不支持锁屏、注销、睡眠状态下控制微信。
- 未代码签名的 EXE 仍可能触发 Windows SmartScreen 提醒。
