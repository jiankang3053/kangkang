# KangkangWeather v3.6.0

## 本轮重点

- 收束最近三轮工程：同类软件调研、P0 稳定性规划、智能提醒后台勿扰升级，形成可构建的新版本。
- 新增智能提醒策略：
  - `always_full`：总是完整提醒。
  - `smart`：按天气正常/异常选择提醒动作。
  - `abnormal_only`：只有异常天气才提醒。
  - `short_daily`：每天只发一句简短天气。
  - `silent`：暂停天气提醒但保留任务记录。
- 新增天气正常跳过发送历史：天气正常且策略为不提醒时，会写入发送历史和自动化状态，避免用户误以为任务坏了。
- 新增勿扰检测：自动发送前检测前台全屏、游戏/视频/演示进程和窗口关键词，忙碌时可延迟、跳过、只记录或强制发送。
- 新增 HKCU 开机自启动管理：不要求管理员权限，兼容源码运行、PyInstaller EXE、中文路径和空格路径。
- 增强托盘后台运行：菜单支持显示主窗口、刷新天气、发送今日天气、暂停/恢复自动化、打开设置、导出诊断包和退出程序。
- Web 控制台和桌面设置页都加入“智能提醒与后台运行 / 勿扰模式”配置。
- 示例配置、README、便携版说明和智能后台说明文档已同步。

## 兼容性

- 默认策略保持“正常天气完整日报”，避免升级后突然少发。用户可在设置里改成“简短一句话”或“不发送”。
- 手动发送和测试消息不受勿扰检测限制；勿扰只拦截自动发送。
- 没有重写微信核心发送链路，真实发送仍走现有 `pywinauto-session` 前台自动化。
- 多对象发送、发送批次、发送历史和全局发送锁继续保留。

## 已知限制

- 官方 Windows 微信没有公开本地发消息 API，锁屏、注销、睡眠、UAC 安全桌面、权限级别不一致时仍不能保证自动发送。
- 全屏/游戏检测不保证 100% 准确，部分游戏或反作弊软件可能阻止进程信息读取。
- 开机自启动可能被 Windows 启动项设置或安全软件禁用。
- 未代码签名的 EXE 仍可能触发 Windows SmartScreen 提醒。
