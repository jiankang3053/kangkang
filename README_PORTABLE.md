# Kangkang Weather 便携版

## 使用步骤

1. 解压 `KangkangWeather-v3.6.0.zip`。
2. 双击 `KangkangWeather.exe`。
3. 首次打开后，在设置页选择省/市/区县并填写微信好友或群名称。
4. 保持官方 Windows 微信客户端已登录。
5. 先点“打开会话测试”，确认能切到目标聊天且不会发送消息。
6. 再点“发送测试消息”做真实发送 smoke test。
7. 确认无误后，再点“发送今日天气”。
8. 在“自动化设置”里配置运行时间段、固定发送点和提醒阈值。
9. 需要时在主界面填写“发送前缀”并保存，完整天气日报会自动加在最前面。

## 常见问题

- 如果 Windows SmartScreen 提醒，这是因为没有代码签名证书，不代表程序逻辑错误。
- 好友或群名称必须和微信搜索结果显示的名称一致。
- “打开会话测试”不会发送消息；“发送测试消息”会真实发送一条短测试消息。
- v3.1.1 搜索好友优先使用 UIAutomation/键盘输入，默认不会点击搜索框坐标或发送按钮坐标。
- v3.5.0 开始记录天气发送批次和每个微信目标的发送结果，避免手动点击和自动任务同时触发造成重复发送。
- v3.6.0 增加智能提醒、后台托盘、开机自启和勿扰检测。天气正常时可选择简短提醒或不提醒，全屏/游戏/演示时可延迟或跳过自动发送。
- 主界面左上角会显示当前版本号；运行总览只做轻量状态展示，深度环境检查需要手动点击对应按钮。
- 正文默认用 Enter/Ctrl+Enter 键盘提交发送；如果微信窗口中途关闭，会停止输入并提示导出诊断包。
- 自动任务只会在配置的运行时间段内发微信；时间段外只记录跳过。
- “天气查询记录”会显示每次查天气的数据源、失败源、缓存状态和耗时。
- 完整天气日报只显示今日分时段、明天和后天；自动补发仍是短提醒。
- 如果 8766 端口被占用，程序会自动换一个可用端口并打开正确页面。
