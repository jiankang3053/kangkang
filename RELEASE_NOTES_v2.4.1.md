# Kangkang Weather v2.4.1

## 重点更新

- 地址配置改为全国省/市/区县三级选择，界面不再要求用户填写经纬度。
- 新增地址本地搜索，支持输入“北京、朝阳区、嘉鱼”等关键字快速选择。
- 新增天气查询记录，最近 200 次查询会记录数据源、失败源、缓存状态、多源分歧和耗时。
- 桌面端自动化设置改为独立可滚动窗口，小窗口下也能操作到底部保存按钮。
- 新增用户级安装包 `KangkangWeatherSetup-v2.4.1.exe`，安装到 `%LOCALAPPDATA%\Programs\KangkangWeather`。

## 交付物

- `KangkangWeather-v2.4.1.zip`：便携版。
- `KangkangWeatherSetup-v2.4.1.exe`：安装版，不需要管理员权限。
- `SHA256SUMS-v2.4.1.txt`：文件校验信息。

## 已知限制

- 仅支持 Windows 10/11 + 官方 Windows 微信客户端，且微信必须已登录。
- 好友或群名称必须和微信搜索结果显示名一致。
- 安装包没有代码签名，Windows SmartScreen 可能提示风险。
