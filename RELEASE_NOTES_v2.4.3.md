# Kangkang Weather v2.4.3

## 重点修复

- 微信搜索目标联系人不再强依赖剪贴板，优先使用 UIAutomation 写入搜索框，失败后再用 Unicode 键盘输入，剪贴板只作为最后兜底。
- 默认禁用搜索框坐标点击和发送按钮坐标点击，减少误点表情、浮层或异常控件的风险。
- 搜索结果必须精确匹配好友/群名称；没有精确匹配时不会按回车盲开会话。
- 发送前会自动扫描并评分选择官方 Windows 微信主窗口，记录窗口标题、class、handle、pid 和 exe 路径。
- 如果微信窗口在搜索或发送过程中关闭/崩溃，程序立即停止后续输入，并返回“微信窗口异常退出或不可访问”诊断。

## 使用建议

- 先用 `文件传输助手` 或 `微信快存` 执行“打开会话测试”和“发送测试消息”。
- 微信和本程序保持同级权限运行，不要一个管理员权限、一个普通权限。
- 失败时导出诊断包，重点查看 `app.log`、微信窗口列表、权限状态、搜索框/输入框状态。
