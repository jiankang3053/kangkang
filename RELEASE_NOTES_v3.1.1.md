# KangkangWeather v3.1.1

## 修复

- 修复桌面版启动时 `/api/state` 可能被微信 UIAutomation、`powercfg` 或 `schtasks` 深度检查拖慢，导致 `timed out` 并弹出 “Failed to execute script” 的问题。
- `/api/state` 改为轻量总览，启动不再主动探测微信窗口；深度检查仍通过“自动发送环境”“息屏/睡眠检查”“定时任务医生”按钮手动触发。
- 桌面窗口标题区同步显示版本号，网页控制台左上角继续显示版本号。

## 说明

- 这个版本的重点是修复 v3.1.0 的启动超时崩溃，不改变微信发送底层逻辑。
- 如果需要判断电脑是否适合息屏自动发送，请进入控制台后手动点“自动发送环境”和“息屏/睡眠检查”。
